# HTML Technical Documentation Page 📄

A fully responsive, interactive HTML & CSS documentation page covering 30+ core HTML concepts with seamless navigation and accessible markup.

---

## 🚀 Features

- **6-section interactive layout** covering tags, attributes, best practices, and more
- **Fixed sidebar** with 100% internal link coverage for instant section access
- **Fully responsive** — tested across all devices from 320px to 1440px screen widths
- **Semantic, accessible HTML** using ARIA-ready tags and proper document structure

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
